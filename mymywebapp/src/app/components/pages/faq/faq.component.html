How do you I register my MyMy Twist Slim Pen?

MyMy TWIST Slim Pen can be registered online on the MyMy Vape website. It is imperative that you register your product. You can only file your warranty claims on the product if it is registered.

Before you register your MyMy TWIST Slim Pen online, you will need to have the following items ready.

  • Serial Number of the product  - Depending on the version of the product you have, you will find the serial number either on the warranty card which came with the product or imprinted on the bottom section of the battery.
  • Picture of the serial number Please make sure you have a picture of legible serial number on either on the warranty card or at the bottom section of the battery.
  • Date of Purchase Please make sure you put in the correct date of purchase. This will be validated when you file for a warranty claim.
  • First, Last name of the purchaser  - Please make sure you put in a valid first and last name of the purchaser. This will be validated while filing a warranty claim
  • Date of Birth We ask this information so that we make sure only adults are able to register products. Yes!! We care.
  • Email of the buyer  - We ask this information so that we can send you a confirmation of registration which you can keep for your records in the future.
  • Address, City, State, ZipCode and Country Details of the buyer  - We ask this information so that we can mail you a warranty coupon code if you request a replacement item. Please make sure that it is the correct address. At this time, we only serve replacements for USA and CANADA.

Once you are ready with the above information, Please click on the link below and enter the information and press the Register button.

http://www.mymyvape.com/pages/register-warranty

Make sure you receive a confirmation message below the button mentioning that your registration request has been received. You should also receive a registration confirmation email in the email you have provided us. If you do not see either of this happening, please contact us at support@mymyvape.com  with the above mentioned information, we will address it as quickly as possible.
